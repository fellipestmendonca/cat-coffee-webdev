# CAT & COFFEE Frontend Project

O projeto foi criado com o objetivo de solucionar a presença online de ONGs.

## Tecnologias utilizadas

JavaScript (ES6+);
HTML 5;
CSS3;

## Instalação

Ao realizar o download do projeto, é recomendado a utilização de um editor de código. Utilizando o Visual Studio Code, por exemplo, é recomendada a extensão "Live Server" por Ritwick Dey.

## Uso

No momento é possível navegar pelas páginas, adicionar informação na área de cadastro, e interagir no site.

## Contribuir

Como primeiro projeto de frontend, aceito qualquer feedback!

Pull requests são muito bem-vindos.

## O que aprendi

Gerenciamento de tempo;

Novas aplicações das tecnologias utilizadas: a utilização de query em JavaScript e melhor entendimento de estruturação semântica de HTML5;
